AI Smart Gym Mirror

Monorepo for the AI Smart Gym Mirror project.

This repository contains three primary workspaces:
- `backend/` - FastAPI backend, database models, WebSocket manager, workers
- `frontend/` - React + TypeScript UI with MediaPipe integration
- `ml/` - Model training, dataset, and model serving tools

Phase 1 focuses on project scaffolding, core session APIs, WebSocket manager, and a minimal MediaPipe streaming demo.

See per-folder READMEs for details.
